Privacy Policy

Last Updated: December 8, 2025

This Privacy Policy outlines the data handling practices of BreatheOSS ("we", "our", "the Project"), a non-profit, open-source initiative. We are committed to transparency and the absolute protection of user privacy.

1. Data Collection

1.1 No Personal Data Collection

We do not collect, store, process, or retain any personally identifiable information (PII). The Service operates without user accounts, cookies, or tracking pixels.

We specifically do not collect:

  • Names, emails, or phone numbers.
  • GPS location data (users manually select regions).
  • IP addresses in any identifiable format.
  • Device identifiers or usage analytics.
1.2 Permissions

The Service requires internet access solely to fetch real-time environmental data. No other device permissions (e.g., storage, location, contacts) are requested.

2. Environmental Data Sources

To provide accurate AQI readings, we query the following public APIs. We transmit only geographic coordinates for the requested zones; no user data is shared.

2.1 Open-Meteo

Used for weather and global air quality modelling. Visit Open-Meteo.

2.2 OpenAQ & CPCB

Used for sensor-based readings in specific regions (e.g., Srinagar). Data originates from the Central Pollution Control Board (CPCB) via OpenAQ.

3. Infrastructure

3.1 Hosting & Security

Our API is hosted on Heroku, and we utilize Cloudflare for DNS and security. While these providers may generate standard technical logs for operational security, BreatheOSS does not access, analyze, or retain these logs to track individual users.

4. Open Source & Licensing

BreatheOSS is open-source software. Our code is publicly available for audit to verify our privacy claims. The project is licensed under the MIT License.

You can inspect the source code at our GitHub Organization.
